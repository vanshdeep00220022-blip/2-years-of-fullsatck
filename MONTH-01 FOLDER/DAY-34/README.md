# Day 34: JavaScript Objects Unlocked 🔓

Welcome to Day 34 of my 2-Year Coding Challenge! Today is all about **JavaScript Objects**. 

## 📝 What I Learned Today
Objects are the core of JavaScript, allowing us to store collections of data and more complex entities. In today's code, I practiced:
* **Object Creation:** Structuring data using key-value pairs.
* **Object Methods:** Writing functions inside objects to perform actions with that object's data.
* **Destructuring:** A clean syntax to extract specific properties from an object into distinct variables.
* **Dynamic Properties:** Adding new keys to an object after it has already been created.
