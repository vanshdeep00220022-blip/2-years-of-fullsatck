## 📝 What I Learned Today

Arrays are special variables that can hold more than one value at a time. They are strictly ordered and incredibly powerful.

### 1. The Zero-Index Rule
The most important concept in computer science formatting: Arrays start counting at `0`, not `1`. 
* `array[0]` gets the 1st item.
* `array[1]` gets the 2nd item.

### 2. Mutability (Connecting back to Day 24 Memory)
Because Arrays are stored in the Heap memory (Reference Types), we can modify their contents even if they are declared with `const`. I practiced updating specific index values directly (e.g., `techStack[1] = "Tailwind"`).

### 3. The Length Property & The "Last Item" Trick
The `.length` property returns the total number of items. Because of zero-indexing, the very last item in an array will always be located at `array.length - 1`.
