# 🚀 100 Days of Code: Full-Stack Developer Challenge

## 📅 Day 41: Advanced Object Manipulation in JS
Building on top of yesterday's fundamentals, Day 41 was all about treating objects like dynamic data structures. I focused on how to extract, loop, and safely copy object data without mutating the original state.

### 📚 Core Concepts Covered:
* **`Object.keys()`**, **`Object.values()`**, and **`Object.entries()`** for data extraction.
* Iterating through key-value pairs using the **`for...in`** loop.
* Cloning objects safely using the modern **Spread Operator (`...`)**.

### 💻 Why this matters for Full-Stack:
When fetching data from a database or a third-party API, the data always arrives as an object or an array of objects. Knowing how to loop through and transform these structures seamlessly is essential backend logic.
