# Day 37: Object Destructuring & JSON 📦🌐

Welcome to Day 37 of my 730-Day Coding Challenge! Today I bridged the gap between raw data and usable code.

## 📝 What I Learned Today
* **JSON (JavaScript Object Notation):** The standard format for storing and transporting data across the web. I learned how it differs slightly from standard JS objects (like requiring double quotes around keys).
* **`JSON.parse()`:** Converting a JSON string from a web server into a JavaScript object.
* **`JSON.stringify()`:** Converting a JavaScript object back into a JSON string to send it to a server.
* **Object Destructuring:** A beautiful ES6 syntax that allows me to unpack properties from objects directly into distinct variables. This keeps my code incredibly clean and readable!
